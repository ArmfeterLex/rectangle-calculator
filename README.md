# Rectangle Calculator

Модуль для консольных приложений на C#, вычисляющий периметр и площадь прямоугольника по двум сторонам с округлением до двух знаков.

## Функциональность

- Ввод двух сторон прямоугольника
- Вычисление периметра: P = (a + b) * 2
- Вычисление площади: S = a * b
- Округление результатов до двух знаков
- Вывод обоих результатов

## Пример работы

Введите стороны прямоугольника
5
3
Периметр прямоугольника 16.00; Площадь прямоугольника 15.00

Введите стороны прямоугольника
4.5
2.5
Периметр прямоугольника 14.00; Площадь прямоугольника 11.25
